Link Wrapper

Servers
https://redirect.partner.fatcoupon.com - Production

default

Method Endpoint Description
GET /go Any click that goes through FatCoupon Partners redirect domain (redirect.partner.fatcoupon.com) in the correct syntax (as described below) will be monetized via the appropriate affiliate program. If FatCoupon Partner can't monetize a given URL it will simply redirect the user to the original URL without modifying it.

Parameters

Name Description Type Example
cid required Your Company Id integer
(query)		 123
mid Merchant Id integer
(query)		 12345
url required This is the URL you are linking to. The URL should always be unencoded. The Link Wrapper will attempt to convert this URL into a monetized URL, and redirect the user there. string
(query)		 https://www.nike.com
clickId This is an optional user specified parameter used to pass your own internal tracking through FatCoupon Partner' monetization engine. string
(query)		 YOUR_TRACK_ID
subId1 Tracking Parameter's which were appended in tracking url. string
(query)		 subId1
subId2 Tracking Parameter's which were appended in tracking url. string
(query)		 subId2
subId3 Tracking Parameter's which were appended in tracking url. string
(query)		 subId3

(query)		 subId4

(query)		 subId5
