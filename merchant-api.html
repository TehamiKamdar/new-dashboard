Merchant API

The Merchant API is the main interface for publishers wishing to know statistics about the merchants on our networks. The Merchant API returns all merchants that FarCoupon works with. Information exposed includes commission rate, conversion rate and domains associated with the merchant. The API can also be used to conduct searches for merchants. The API uses RESTful calls and responses are formatted in JSON.

Search Merchants
GET

Returns merchants based on search criteria

Endpoint
https://api.partner.farcoupon.com/api/stores
Parameters
Parameter Type Required Description
search string No Search query to find merchants by name
cashbackType string No Type of cashback offered
page integer No Page number for pagination
Example Request
curl -X 'GET' \
"https://api.partner.farcoupon.com/api/stores?search=nike&cashbackType=0&page=1&size=30" \
-H "accept: application/json"
Responses
Status Code Description
200 Success - The request was successful
400 Bad Request - Invalid syntax or parameters
401 Unauthorized - Authentication required
403 Forbidden - Insufficient permissions
404 Not Found - Resource does not exist
500 Internal Server Error - Unexpected server error
